AHOI API: erste Schritte

AHOI API: first Steps

Um auf Kontodaten zugreifen zu können, muss ein Benutzer zuerst einmal einen Zugriff auf seine Bankdaten erlauben. Üblicherweise wird dazu die PIN und der Benutzername benötigt.

In order to access account data, a user must first grant access to his bank(s). Usually, the PIN and the user name are required.

Die Erlaubnis, auf die Bankdaten über die AHOI API zugreifen zu dürfen, muss nur einmal erteilt werden. Sie ist solange gültig, bis sie ausdrücklich widerufen wird.

The permission to access the bank data via the AHOI API must be granted only once. It is valid until an user revokes permission.

Auf den nachfolgenden Seiten werden die einzelnen Schritte demonstriert und die technischen Hintergründe kurz erläutert.

On the following pages, the steps from bank selection to transaction display will be demonstrated and the technical background shortly explained.

Folgende Schritte werden gezeigt:

The following steps are shown:

  1. Neuen Benutzer in AHOI anlegen
  2. Create a new user in AHOI
  3. Bank auswählen
  4. Select bank
  5. Zugriff auf Bank-Konten erlauben
  6. Grant access to bank accounts
  7. Konten für Anzeige auswählen (optional)
  8. Select accounts for display (optional)
  9. Kontoein- und ausgänge anzeigen
  10. Display account balances and transactions

Hat ein Benutzer Konten bei mehreren Banken, müssen die Schritte 2 - 4 für jede Bank einmal wiederholt werden.

If a user has accounts with multiple banks, steps 2 - 4 must be performed again for each bank
